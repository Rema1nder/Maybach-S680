# 蓝牙指令协议 (BTComm)

> 源文件：`User/BTComm.h` / `User/BTComm.c`  
> 串口：USART2，引脚 PA2 (TX) / PA3 (RX)，波特率在 `Uart2_Init()` 中配置

---

## 一、指令格式

```
CMD\n
CMD:参数\n
```

- 命令与参数之间用 `:` 分隔，换行符（`\r` 或 `\n`）标志指令结束。
- 命令大小写不敏感（接收时统一转为大写）。
- 缓冲区大小 **64 字节**，超出则丢弃当前命令并重新接收。
- 每条指令执行后会返回一条以 `OK:` 或 `ERR:` 开头的响应字符串。

---

## 二、工作模式

系统存在三个互斥模式标志，影响部分命令的生效范围：

| 标志 | 含义 |
|---|---|
| 急停 (`ESTOP`) | 所有电机立即停止，方向控制指令被拒绝 |
| 键控模式 (`KEY:1`) | 蓝牙遥控驱动电机方向，退出巡线模式 |
| 巡线模式 (硬件 K1) | 自动巡线，蓝牙键控指令被拒绝 |

> 硬件按键 K2 与 `KEY:1/0` 指令等价，二者可互相切换键控模式。

---

## 附录：硬件按键功能

| 按键 | 引脚 | 功能 | 说明 |
|---|---|---|---|
| **K1** | PA5 | 巡线模式 启/停 | 键控模式激活时无效；**启动流程**：LED 变黄 → 陀螺仪静止校准 2s（请保持静止）→ 校准完成后 LED 变绿、清除急停、复位圆环和里程计、启动负压风扇（转速取蓝牙设置值）并开始巡线；停止时电机和负压全部关停、LED 熄灭；急停（K4）期间可取消待校准状态 |
| **K2** | PA4 | 蓝牙键控模式 启/停 | 巡线模式激活时无效；进入时自动清除急停、启用电机和轮子锁定、LED 蓝色，与蓝牙指令 `KEY:1/0` 等价；退出时停止电机并解锁轮子 |
| **K3** | PC14 | 光电管阈值校准 | **第一次按**：开始校准，LED 变黄，小车在黑白赛道上扫一遍；**第二次按**：结束校准，自动计算 16 路阈值 `= (min+max)/2` 并应用，LED 短绿后熄灭，串口输出各路 min/max/threshold |
| **K4** | PC13 | 急停 / 解除急停（切换） | 非急停状态下按下触发急停（停止所有电机/负压、退出所有模式、LED 红色）；急停状态下按下清除急停、LED 熄灭 |

> **互斥规则：**
> - K1 在键控模式激活时按下无效
> - K2 在巡线模式运行时按下无效
> - K4（急停/解除）任何时候均有效

## 三、全局指令（任何时候均生效）

### 3.1 急停与恢复

| 指令 | 简写 | 响应 | 说明 |
|---|---|---|---|
| `ESTOP` | `E` | `OK:ESTOP` | 急停：停止所有电机和负压，退出所有模式，LED 变红 |
| `CLR` | `CLEAR` | `OK:ESTOP_CLR` | 清除急停状态，LED 熄灭 |

**急停触发效果：**
- 所有驱动轮立即停止，禁用电机驱动
- 负压电机 PWM 置零并停止
- 退出巡线模式 (`star_car = 0`)
- 退出键控模式
- 重置圆环状态机

---

### 3.2 负压风扇控制

| 指令 | 响应 | 说明 |
|---|---|---|
| `VON` | `OK:VON` | 开启负压电机（使用当前已设置的转速） |
| `VOFF` | `OK:VOFF` | 关闭负压电机 |
| `VS:xx` | `OK:VS=xx` | 设置负压风扇转速，`xx` 范围 **0–100**（百分比），若电机已开启则立即生效 |

> 注意：`VON` 时若当前转速为 0，则默认使用内部默认值（当前代码配置为 0%，需要先发 `VS:xx` 设置转速）。

---

### 3.3 基础速度

| 指令 | 响应 | 说明 |
|---|---|---|
| `SPD:xx` | `OK:SPD=xx` | 设置键控模式下的默认速度，`xx` 范围 **0–250**，默认值为 30 |

---

### 3.4 电池电压查询

| 指令 | 响应格式 | 说明 |
|---|---|---|
| `BATV` | `BATV:x.xxV` | 查询当前电池电压（实时读取 ADC 折算值） |

---

### 3.5 键控模式开关

| 指令 | 响应 | 说明 |
|---|---|---|
| `KEY:1` | `OK:KEY=ON` | 进入键控模式，清除急停，停止运动，退出巡线，LED 变蓝 |
| `KEY:0` | `OK:KEY=OFF` | 退出键控模式，停止运动，LED 熄灭 |

---

### 3.6 PID 参数整定

| 指令 | 格式 | 响应 | 说明 |
|---|---|---|---|
| `PID` | `PID:kp,ki,kd` | `OK:PID=kp,ki,kd` | 同时设置左右轮速度 PID |
| `PID` | `PID:kp,ki,kd,kf` | `OK:PID=kp,ki,kd,KF=kf` | 同上，附带前馈系数 |
| `KF` | `KF:xx` | `OK:KF=xx` | 单独设置左右轮前馈系数 |
| `PPID` | `PPID:kp,ki,kd,gyro_kd` | `OK:PPID=kp,ki,kd,gyro_kd` | 设置位置环 PID 及陀螺仪阻尼 |
| `GYRO` | `GYRO:1` / `GYRO:0` | `OK:GYRO=ON/OFF` | 开启/关闭陀螺仪阻尼（架空测试时建议关闭） |
| `ISPD` | `ISPD:base,range` | `OK:ISPD=base,range` | 设置巡线速度参数（基础速度和范围） |
| `ISPD` | `ISPD`（无参数） | `ISPD:base,range` | 查询当前巡线速度参数 |

---

### 3.7 陀螺仪校准与数据流

| 指令 | 响应 | 说明 |
|---|---|---|
| `GCAL` | `GYRO:CAL` → (2s 后) `GYRO:DONE` → 持续输出数据帧 | 静止校准陀螺仪 2s（1000 × 2ms），完成后以 **5 Hz** 持续输出 yaw 角、过滤后 gz、原始 gz |
| `GSTOP` | `GYRO:STOP` | 停止陀螺仪数据流输出 |

**GCAL 数据帧格式（每 200 ms 一帧）：**

```
YAW:xx.xx,GZF:xx.xx,GZR:xx.xx
```

| 字段 | 说明 |
|---|---|
| `YAW` | 当前偏航角（deg），梯形积分累积值 |
| `GZF` | 经死区滤波（±0.8 dps）后的 gz（dps） |
| `GZR` | 仅去零偏、未加死区的原始 gz（dps） |

> **注意：** 校准期间（约 2s）小车必须保持静止；GCAL 独立于巡线/键控模式之外，任何时候均可发送。

---

### 3.8 黑线传感器模拟

| 指令 | 格式 | 响应 | 说明 |
|---|---|---|---|
| `BLK` | `BLK:ch1,ch2,...` | `OK:BLK=N ch` | 模拟指定通道（0–15）检测到黑线，用于脱线调试 |
| `BLK` | `BLK:OFF` / `BLK:0` | `OK:BLK=OFF` | 关闭模拟模式，恢复真实传感器数据 |
| `BLK` | `BLK`（无参数） | `OK:BLK=7,8(default)` | 启用默认模拟（中间两路，通道 7 和 8） |

---

### 3.9 光电管实时数据流

| 指令 | 响应 | 说明 |
|---|---|---|
| `LSEN` | `OK:LSEN` → 持续输出数据帧 | 开启 16 路光电管黑线检测数据流，以 **5 Hz**（每 200 ms）持续输出 |
| `LSTOP` | `LSEN:STOP` | 停止数据流输出 |

**LSEN 数据帧格式（每 200 ms 一帧）：**

```
BLK:xxxxxxxxxxxxxxxx
```

| 字段 | 说明 |
|---|---|
| `BLK:` | 固定前缀 |
| 16 位字符串 | 每位对应一路光电管（通道 0–15），`1` = 检测到黑线，`0` = 白色/无黑线 |

**示例：**
```
BLK:0000000110000000   ← 通道 7、8 检测到黑线（小车压中线）
BLK:1111111111111111   ← 全部 16 路检测到黑线
BLK:0000000000000000   ← 全部 16 路未检测到黑线
```

> LSEN 独立于巡线/键控模式之外，任何时候均可发送。  
> 可与 `GCAL` 同时运行，两路数据流互不干扰。

## 四、查询指令

| 指令 | 简写 | 响应格式 | 说明 |
|---|---|---|---|
| `?STATUS` | `STATUS` / `?` | `MODE:xx,SPD:xx,BASE:xx,VAC:xx,VS:xx,KEY:xx,ESTOP:xx` | 查询完整运行状态 |
| `?ADC` | `ADC` | `ADC[0-7]:...` / `ADC[8-15]:...` / `BLK:...` | 查询 16 路光电管 ADC 值及黑线检测状态 |
| `?PID` | — | `TGT/ACT/PWM` 三行 | 查询左右轮目标速度、实际速度、PWM 输出 |
| `?ODOM` | `ODOM` | 里程/坐标/航向/编码器计数 | 查询里程计数据 |
| `?CALIB` | — | `CALIB:pulses=xx (active/inactive)` | 查询当前编码器校准状态 |
| `HELP` | `H` | 命令速查表 | 返回简要帮助 |

### `?STATUS` 响应字段说明

| 字段 | 取值 | 含义 |
|---|---|---|
| `MODE` | `IDLE` / `FWD` / `BWD` / `TL` / `TR` | 当前运动模式 |
| `SPD` | 0–250 | 当前运动速度 |
| `BASE` | 0–250 | 基础速度（默认 30） |
| `VAC` | `ON` / `OFF` | 负压是否开启 |
| `VS` | 0–100 | 负压风扇转速百分比 |
| `KEY` | `ON` / `OFF` | 键控模式状态 |
| `ESTOP` | `YES` / `NO` | 急停状态 |

## 五、编码器校准流程

```
1. 发送 CALIB_START   → 开始记录脉冲，LED 亮黄色
2. 手动将小车在直线上推动已知距离（如 1000 mm）
3. 发送 CALIB_END:1000 → 传入实际推动距离(mm)
```

| 指令 | 响应 | 说明 |
|---|---|---|
| `CALIB_START` / `CALIB` | `OK:CALIB_START` | 开始校准，LED 变黄 |
| `CALIB_END:dist` | `CALIB:pulses=xx` / `NEW_PULSE_PER_MM=xx` | 结束校准，计算新的脉冲/mm 系数 |
| `CALIB_END`（无参数） | 按 500/1000 mm 推算三组参考值 | 仅显示脉冲数，需手动填写距离 |

> 校准完成后根据返回的 `NEW_PULSE_PER_MM` 值更新 `Odometer.h` 中的 `ODOM_ENCODER_PULSE_PER_MM`。

---

### 里程计相关指令

| 指令 | 响应 | 说明 |
|---|---|---|
| `ODOM_RST` | `OK:ODOM_RST` | 重置坐标、航向及累计里程 |
| `ODOM_ZERO` | `OK:ODOM_ZERO` | 仅重置坐标和航向，保留累计里程 |

## 六、键控模式下的运动指令

> **前提：** 已通过 `KEY:1` 或硬件 K2 进入键控模式，且未处于急停状态。  
> 若未进入键控模式发送方向指令，返回 `ERR:NOT_KEY_MODE`。  
> 若处于急停状态发送方向指令，返回 `ERR:ESTOP_ACTIVE`。

| 指令 | 简写 | 响应 | 说明 |
|---|---|---|---|
| `FWD` | `W` | `OK:FWD=xx` | 前进（速度使用基础速度） |
| `FWD:xx` | — | `OK:FWD=xx` | 前进，速度 `xx`（0–100） |
| `BWD` | `S` | `OK:BWD=xx` | 后退 |
| `BWD:xx` | — | `OK:BWD=xx` | 后退，速度 `xx` |
| `TL` | `A` | `OK:TL=xx` | 原地左转 |
| `TL:xx` | — | `OK:TL=xx` | 左转，速度 `xx` |
| `TR` | `D` | `OK:TR=xx` | 原地右转 |
| `TR:xx` | — | `OK:TR=xx` | 右转，速度 `xx` |
| `STOP` | `X` | `OK:STOP` | 停止移动（保留键控模式） |

## 七、错误响应一览

| 响应 | 触发条件 |
|---|---|
| `ERR:ESTOP_ACTIVE` | 急停中发送方向指令 |
| `ERR:NOT_KEY_MODE` | 非键控模式下发送方向指令 |
| `ERR:PID_FMT` | `PID` 指令参数格式错误（需 3 或 4 个浮点数） |
| `ERR:PPID_FMT` | `PPID` 指令需 4 个浮点数 |
| `ERR:KF_FMT` | `KF` 指令参数格式错误 |
| `ERR:BLK_FMT` | `BLK` 指令通道解析失败 |
| `ERR:ISPD_FMT` | `ISPD` 需 2 个参数 |
| `ERR:NOT_CALIB` | 未启动校准时发送 `CALIB_END` |
| `ERR:UNKNOWN_CMD` | 无法识别的命令 |
| `GYRO:CAL` | 发送 `GCAL` 后立即返回，表示校准已开始 |
| `GYRO:DONE` | 校准完成（约 2s 后），随即开始数据流 |
| `GYRO:STOP` | 发送 `GSTOP` 后返回，数据流已停止 |
| `OK:LSEN` | 发送 `LSEN` 后立即返回，表示数据流已开启 |
| `LSEN:STOP` | 发送 `LSTOP` 后返回，数据流已停止 |

## 八、快速参考

```
# 急停/恢复
ESTOP / CLR

# 负压控制
VS:80        → 设置负压转速 80%
VON          → 开启负压
VOFF         → 关闭负压

# 遥控操作
KEY:1        → 进入遥控模式
SPD:50       → 默认速度 50%
FWD          → 前进（或 W）
FWD:60       → 以 60% 速度前进
BWD / TL / TR / STOP  → 后退/左转/右转/停止
KEY:0        → 退出遥控模式

# 状态查询
?            → 完整状态
?ADC         → 传感器原始值
BATV         → 电池电压
?PID         → PID 运行状态
?ODOM        → 里程计数据

# PID 整定
PID:1.2,0.01,0.05
PID:1.2,0.01,0.05,0.3   → 带前馈
GYRO:0                   → 架空调试时关闭陀螺仪阻尼

# 陀螺仪调试
GCAL         → 静止 2s 校准零漂，随后以 5Hz 输出 yaw/gz 数据流
GSTOP        → 停止数据流

# 光电管数据流
LSEN         → 以 5Hz 输出 16 路光电管黑线检测结果（BLK:xxxxxxxxxxxxxxxx）
LSTOP        → 停止光电管数据流
```
